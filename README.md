# Taller 01 - Análisis de Procrastinación

## Descripción del Proyecto
Este proyecto es un software por consola desarrollado en Java que permite a los usuarios ingresar con sus credenciales para registrar, modificar y eliminar sus horas de actividades. Además, incluye un sistema de análisis de datos de los archivos de texto para calcular las estadisticas de procrastinación, como la actividad más frecuente y el usuario con más horas invertidas.

## Instrucciones de Ejecución
1. Clonar el repositorio en el equipo local.
2. Abrir el proyecto utilizando el IDE Eclipse.
3. Verificar que los archivos `Usuarios.txt` y `Registros.txt` se encuentren en la base del proyecto.
4. Compilar y ejecutar la clase `taller.java`.
5. Utilizar el teclado para escoger la actividad deseada en la consola.
